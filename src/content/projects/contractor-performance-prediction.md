title: "A Data-Driven Decision Support Tool for Contractor Performance Prediction in Lean Construction Projects"
status: "Active Work"
summary: "This project proposes a machine learning-based decision support tool to predict contractor performance in lean construction projects. The approach uses historical audit data, contractor profile characteristics, KPI performance records, and work package information to estimate expected contractor performance for future project scopes. By integrating data-driven predictive models with scope similarity analysis, the tool supports more informed contractor selection and assignment decisions. The objective is to help project managers reduce uncertainty, improve planning, and promote better performance outcomes in construction projects by leveraging existing project data."
