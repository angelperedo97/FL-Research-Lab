## Project Overview

Currently researching GPU systems and ML systems methods for workload-aware CUDA kernel/configuration selection. This work builds on prior GPU benchmarking and parallel sorting work, with a focus on lightweight workload features, adaptive runtime selection, tail-latency behavior, and robustness under workload distribution shift.
